# Turbina-Turbojet-Inventor3D
Simplified Turbojet Turbine that i made and dimensioned the components for. It follows the brayton Joule cycle and i designed it without oleodinamic components other than Squeeze Film Dumpers, necessary to reduce vibrations produced by the turbine.
Just open it with inventor3D and everithingh should be fine.

If inventor says that there may be a componenet missing, just use another bearing.
